   - 모두가 부담 없이 동아리를 접할 수 있고 각 동아리를 쉽고 편리하게 관리할 수 있도록 하기 위함
   - 동아대학교 학생 정보를 이용해서 로그인 -> 간단한 동아리 생성 or 가입 -> 해당 동아리에서 커뮤니케이션 활동
